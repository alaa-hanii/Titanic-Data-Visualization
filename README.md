📌 Overview
This project explores the Titanic dataset to analyze passenger survival patterns using Python data visualization tools.
The objective is to identify key factors that influenced survival rates and present insights through visual storytelling.

📊 Dataset
The dataset contains information about passengers such as:
Age
Gender
Passenger Class
Fare
Survival Status

🛠 Tools & Technologies
Python
Pandas
Matplotlib
Seaborn
Jupyter Notebook

📈 Key Insights
Female passengers had significantly higher survival rates.
First-class passengers were more likely to survive.
Age distribution affected survival probability.
